# ADR 001：本地优先 Web App

## 状态

已接受。

## 背景

GPT Image Studio 的目标是在没有自建后端的情况下也能使用。应用会产生会话、消息、生成或导入的图片，以及用户设置，这些数据都属于用户的本地工作区。

## 决策

Web App 继续保持本地优先：

- 会话、消息、图片元数据、图片 Blob 和设置保存在浏览器 IndexedDB。
- 轻量草稿在迁移到对话级草稿前，可以继续使用 localStorage。
- 应用可以直接从浏览器调用用户配置的 OpenAI 兼容 Images API。
- 应用提供备份导出和恢复，而不是云同步。

## 影响

- 用户不需要安装服务端也能使用应用。
- 数据不经过 GPT Image Studio 自己的后端。
- 浏览器存储上限和 CORS 仍然是产品边界。
- 保存在浏览器存储里的 API key 适合个人设备，不适合公共设备。
- 更敏感的账号能力应该交给可选的本地 companion，而不是把 OAuth 凭据保存在浏览器里。
